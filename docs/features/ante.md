# アンティ設定

**ステータス**: `planned`

## 概要

各ハンド開始時に全プレイヤーが強制的に一定額を支払うアンティを設定できるようにする。

## 動機

- アンティありのゲーム形式に対応する
- ポットが大きくなりやすくアクションが活発になる

## 主要な機能

- テーブル作成時にアンティ額を設定（0 でアンティなし）
- 毎ハンド開始時、全プレイヤーのスタックからアンティを徴収してポットへ
- スタック不足のプレイヤーはオールインアンティ（持ち金全額）

## 技術的な検討事項

- `TableRecord` / `CreateRingTableInput` に `ante` フィールドを追加
- `GameState.startHand()` でブラインドポスト前にアンティを徴収
- `GameProperties` にデフォルトアンティ額を追加（デフォルト 0）
- `GameResponse` に `ante` を含めてフロントに表示
