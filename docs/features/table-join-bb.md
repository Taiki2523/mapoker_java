# テーブル参加時の 1BB 強制支払い

**ステータス**: `planned`

## 概要

テーブルに途中参加したプレイヤーは、最初のハンドで 1BB を強制的に支払ってからゲームに参加できるようにする。

## 動機

- ブラインドポジションを待たずに有利なポジションから参加する「エントリー待ち」を防ぐ
- 標準的なカジノルールに準拠する

## 主要な機能

- 途中参加（ディーラーボタンの左隣以外のポジションで着席）した場合、次ハンド開始時に 1BB を強制ポスト
- SB/BB と同じポジションで着席した場合は通常のブラインドのみ
- ポスト済みプレイヤーはプリフロップでアクション権を持つ（BB と同様）

## 技術的な検討事項

- `Player` に `mustPostBb` フラグを追加
- `startHand()` でフラグを確認し、通常ブラインドに追加してポスト処理
- フロントエンドで参加時に「1BB をポストしてゲームに参加」の確認表示（任意）
