# 観戦モード

**ステータス**: `planned`

## 概要

テーブルに参加せずにゲームを観戦できるモードを実装する。

## 動機

- 満席のテーブルでも待ちながら観戦できる
- 学習目的でプレイを観察したいユーザーへの対応

## 主要な機能

- テーブルに観戦者としてロビーから参加
- 全プレイヤーのホールカードが見える観戦者専用ビュー（ゲーム終了後または設定により）
- 観戦者一覧の表示

## 技術的な検討事項

- `viewer_index` クエリパラメータが `-1`（観戦者）の場合、ショーダウン時のみ全カードを返すよう `GameResponse` を調整
- 観戦者のセッション管理（通常の着席とは分離）

## 依存する先行作業

- なし
