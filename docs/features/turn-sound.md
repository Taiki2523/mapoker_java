# 手番通知サウンド

**ステータス**: `planned`

## 概要

自分の手番が来たときにサウンドを鳴らし、よそ見していてもアクションを促せるようにする。

## 動機

- 別タブや別アプリを操作中に手番が来ても気づけない
- 通知音があることでゲームの没入感も高まる

## 主要な機能

- 自分のターンになった瞬間にサウンドを再生
- サウンドのオン/オフをユーザーが設定できる（初回は off にしてブラウザ制限を回避）
- 音量調整（任意）

## 技術的な検討事項

- `isMyTurn` が false → true に変化したタイミングで `Audio.play()` を呼ぶ（App.tsx の useEffect）
- ブラウザはユーザー操作なしのオートプレイを制限するため、初回はユーザーが一度クリックした後に有効化
- 音源ファイルを `mapoker-frontend/public/sounds/` に配置（短い効果音 wav/mp3）
