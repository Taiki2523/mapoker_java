# ハンド履歴ビューアー

**ステータス**: `planned`

## 概要

終了したハンドをアクション単位で再生・確認できるビューアーを実装する。

## 動機

- プレイを振り返ることで戦略の改善につながる
- 相手のベット行動や自分のミスを分析できる

## 主要な機能

- ハンド一覧から任意のハンドを選択
- アクション順にボード・スタック・ベット額が変化するアニメーション再生
- 各アクション時点でのポットオッズ・スタック比率の表示
- 「次のアクション」「前のアクション」のステップ移動

## 技術的な検討事項

- `actions` テーブルに保存された全アクションを順に適用して状態を再構築するリプレイエンジンが必要
- フロントエンドは現在の `GameState` ビューをほぼ流用できる
- ホールカードの公開タイミング（ショーダウンのみ公開）

## 依存する先行作業

- `actions` テーブルへの完全な情報記録（現時点の確認が必要）
