# トーナメントモード

**ステータス**: `planned`

## 概要

現在実装されているキャッシュゲーム（リングゲーム）に加え、参加費を払って一定人数が競うトーナメント形式のゲームモードを追加する。

## 動機

- キャッシュゲームは自由な参加・離脱が可能だが、競技性が低い
- トーナメントは明確な勝者と順位があり、ゲームとしての盛り上がりが生まれる

## 主要な機能

- エントリー費とスターティングスタックの設定
- ブラインド構造の自動アップ（ラウンド制）
- バブル（賞金圏外最後の一人）の管理
- アドオン・リバイオプション（オプション）
- 最終的な順位・賞金の確定と配分

## 技術的な検討事項

- `GameState` にトーナメント用フィールドを追加するか、別エンティティで管理するか
- ブラインドスケジュールの設定フォーマット（JSON カラム）
- 複数テーブルに分散したプレイヤーをどう管理するか（マルチテーブルトーナメント）

## 依存する先行作業

- なし（リングゲームの実装が安定していること）
