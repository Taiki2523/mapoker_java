# mapoker — UI Design Requirements

---

## 概要

2人対戦のテキサスホールデムポーカーWebアプリ。  
PCブラウザでの対戦を想定。URLを共有して2人がそれぞれ自分の画面でプレイする。

---

## 画面一覧

| 画面 | URL | 説明 |
|------|-----|------|
| ロビー | `/` | ゲームの作成・参加 |
| ゲームテーブル | `/?gameId=xxx` | メインのプレイ画面 |

---

## ロビー画面

シンプルな縦並びレイアウト。

```
┌────────────────────────────┐
│                            │
│         mapoker            │  ← ロゴ・タイトル
│                            │
│    [ 新規ゲームを作成 ]      │  ← ボタン
│                            │
│    ゲームIDで参加:           │
│    [ ____________ ] [参加]  │  ← テキスト入力 + ボタン
│                            │
└────────────────────────────┘
```

---

## ゲームテーブル画面

メインの実装対象。以下の要素で構成される。

---

### レイアウト全体

```
┌──────────────────────────────────────────┐
│  ステータスバー（画面上部・固定）            │
├──────────────────────────────────────────┤
│                                          │
│         [ 相手のプレイヤーシート ]          │
│                                          │
│   ┌──────────────────────────────┐       │
│   │                              │       │
│   │     コミュニティカード         │       │
│   │          ポット               │       │
│   │                              │       │
│   └──────────────────────────────┘       │
│              ← テーブル →                │
│         [ 自分のプレイヤーシート ]          │
│                                          │
├──────────────────────────────────────────┤
│  アクションパネル（画面下部・固定）          │
└──────────────────────────────────────────┘
```

テーブルは横長の楕円形。画面中央に配置する。

---

### ステータスバー

画面上部に常時表示される細いバー。

**表示内容（左から順に）：**
- ロゴ（`mapoker`）
- 現在のフェーズバッジ（`PREFLOP` / `FLOP` / `TURN` / `RIVER`）
- ポット額（例：`Pot 350`）
- ベット額（例：`Bet 150`）
- ゲーム状態テキスト（例：`あなたのターン` / `相手のターン` / `終了`）
- URLコピーボタン・設定アイコン（右端）

**フェーズバッジの色分け：**

| フェーズ | 色 |
|---------|-----|
| PREFLOP | グレー |
| FLOP    | ブルー |
| TURN    | オレンジ |
| RIVER   | レッド |

---

### テーブル

画面の中心に配置される楕円形のテーブル。

- 外側はダークブラウンのウッド調の縁（太め）
- 内側はグリーンのフェルト面。中央に向かって少し暗くなるグラデーション
- フェルト面にはうっすらとテクスチャ感がある

---

### プレイヤーシート

テーブルの上下にそれぞれ配置される。  
自分は下、相手は上。

**シートの構成（上から順）：**

```
┌─────────────────────┐
│ [アバター]  name [D] │  ← アバター画像 + プレイヤー名 + ポジションバッジ
│            Stack 850 │  ← スタック額
│         [カード][カード]│  ← 手札2枚
└─────────────────────┘
```

**ポジションバッジ：**  
`D`（ディーラー）/ `SB`（スモールブラインド）/ `BB`（ビッグブラインド）を小さいバッジで表示。

**アバター：**  
丸形。未設定の場合はプレイヤー名の頭文字を表示するフォールバック。

**アクティブ状態（自分のターン）：**  
シート全体がゴールドに光る枠線＋発光エフェクト。ゆっくりと点滅（パルス）する。

**フォールド状態：**  
シート全体が半透明になりグレーアウトする。

**ベット額の表示：**  
シートとテーブル中央の間に、ベット額をチップアイコン＋数値で表示する。  
金額に応じてチップの色が変わる（小額：白、中額：緑、高額：黒、最高額：紫）。

---

### 手札（カード）

**表面：**
- 白背景
- 左上と右下にランク＋スートを小さく表示
- 中央に大きいスートシンボルを表示
- 赤スート（♥ ♦）は赤色、黒スート（♠ ♣）は黒色

**裏面：**
- ポーカーらしい柄（例：赤×紺のパターン）

**相手の手札：**  
ゲーム中は裏面で表示。ショーダウン時にのみ表向きになる。

---

### コミュニティカード

テーブル中央に横一列で配置。最大5枚（フロップ3枚・ターン1枚・リバー1枚）。

- 手札よりひとまわり大きいサイズ
- まだ開かれていないスロットは、薄い破線の枠のみ表示する

---

### ポット

コミュニティカードの上部中央に表示。

- チップアイコン＋`Pot: 350` の形式
- ゴールドのテキスト
- サイドポットがある場合は複数表示

---

### アクションパネル

自分のターン時のみ画面下部に表示される。非ターン時は非表示。

**構成：**

```
┌─────────────────────────────────────────────────────┐
│  手役表示: ワンペア ♥4 ♠9          スタック: 850      │
├─────────────────────────────────────────────────────┤
│  [━━━━━●──────────]  150    [最小] [1/2P] [P] [ALL]  │  ← スライダー
├─────────────────────────────────────────────────────┤
│      [フォールド]     [チェック/コール]   [ベット/レイズ]  │  ← ボタン
└─────────────────────────────────────────────────────┘
```

**ボタンの色分け：**

| ボタン | 色 |
|-------|-----|
| フォールド | ダークレッド |
| チェック / コール | ダークグリーン |
| ベット / レイズ | ダークブルー |

コール時はボタンに金額を表示（例：`コール 150`）。  
レイズ時も同様（例：`レイズ 300`）。

**ベットスライダー：**  
最小ベット〜オールインの範囲で調整できる。  
クイックボタン（最小 / 1/2ポット / ポット / オールイン）で即セットできる。

**手役表示：**  
自分の手札＋コミュニティカードから現在の最強手役をリアルタイムで表示。

---

## アニメーション

### カードディール（ゲーム開始時）

テーブル中央のデッキから、相手→自分の順に1枚ずつ滑るように配られる。  
4枚の配布が0.5秒程度で完了する。

### コミュニティカード開示

- フロップ：3枚が連続してめくられる
- ターン・リバー：1枚ずつめくられる  
カードが裏面から表面にフリップする動作。

### ベット時

プレイヤーのシート付近からチップがテーブル中央（ポット）に向かって移動する。

### アクティブターン

自分のターンになったとき、アクションパネルが下から滑り上がって表示される。

### ショーダウン（勝負）

1. 両者の手札が順にフリップして表向きになる
2. 手役名がテーブル中央にポップイン表示
3. 勝者のシートにゴールドのグロー
4. ポットのチップが勝者に向かって移動する
5. スタックに `+350` のフローティングテキストが出てフェードアウト

---

## 勝利・ゲーム終了

ショーダウンアニメ完了後、テーブル上に結果を表示する。

```
┌───────────────────────────┐
│  🏆 user2 の勝利！          │
│     ワンペア               │
│                           │
│     [ 次のゲームへ ]        │
└───────────────────────────┘
```

- 背景はテーブルを薄暗くするオーバーレイ
- テキストはゴールド
- 「次のゲームへ」ボタンで同じゲームIDのまま次ラウンドへ

---

## カラーイメージ

| 用途 | イメージ |
|------|---------|
| テーブルフェルト | 深いグリーン |
| テーブル縁 | ダークブラウン |
| UI背景・パネル | 半透明ブラック（ガラス質） |
| 強調・アクティブ | ゴールド |
| テキスト | オフホワイト |
| 数値（スタック・ポット） | 等幅フォント |

全体的にカジノらしい重厚感のあるダークテーマ。

---

## 状態別の画面挙動まとめ

| 状態 | 画面の変化 |
|------|-----------|
| 自分のターン | アクションパネル表示、自シートがゴールドに光る |
| 相手のターン | アクションパネル非表示、相手シートがゴールドに光る |
| フォールド済み | フォールドしたシートがグレーアウト |
| ショーダウン | 両者の手札が表向きになり演出が流れる |
| ゲーム終了 | 結果オーバーレイが表示される |
| 相手が切断 | 「相手が切断しました」のメッセージを表示 |
